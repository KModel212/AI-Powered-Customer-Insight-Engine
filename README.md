# AI-POWERED-CUSTOMER-INSIGHT-ENGINE

This project focuses on **analyzing customer reviews** using **Natural Language Processing (NLP), Retrieval-Augmented Generation (RAG), and Large Language Models (LLMs)** to generate actionable business insights and recommendations.

---

## 🛠️ Tools & Technologies

* **Python** – main programming language
* **Pandas / NumPy** – data manipulation and feature engineering
* **Scikit-learn** – sentiment evaluation and TF-IDF retrieval
* **Hugging Face Transformers** – transformer-based NLP models
* **SentenceTransformers + FAISS** – semantic search and lightweight RAG pipeline
* **TextBlob** – baseline sentiment analysis
* **Matplotlib / WordCloud** – data visualization
* **Jupyter Notebook** – interactive development environment

---

## 📌 Project Overview

1. **Data Loading & Exploration** – load public customer review datasets, inspect data quality, and explore key statistics.
2. **Data Cleaning & Feature Engineering** – preprocess review text, remove noise, and extract features such as review length, word count, and sentence count.
3. **Sentiment Analysis & Evaluation** – perform sentiment analysis and evaluate model performance using Accuracy, Precision, Recall, F1-score, and Confusion Matrix.
4. **Semantic Retrieval (Lightweight RAG)** – retrieve the most relevant review chunks using SentenceTransformers + FAISS with a TF-IDF fallback.
5. **Prompt Engineering & LLM Analysis** – generate structured business insights, customer pain points, strengths, weaknesses, and actionable recommendations using LLMs.
6. **Business Insight Report** – summarize findings into an executive-style report to support business decision-making.
